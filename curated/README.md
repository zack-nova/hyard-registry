# Curated Package Handles

Unnamespaced Package Handles live here.

```text
curated/index.yaml
```

These handles are reserved for official or curated packages. Each curated handle
points at a namespaced Package Handle and does not copy full version locator
metadata.

## Current Handles

- `docs`: Docs Orbit from `zack-nova/hyard-demo-docs-orbit`
- `review`: Review Orbit from `zack-nova/hyard-demo-review-orbit`
- `release`: Release Orbit from `zack-nova/hyard-demo-release-orbit`
- `product-lab`: Product Lab Harness Package from `zack-nova/hyard-demo-product-lab`
